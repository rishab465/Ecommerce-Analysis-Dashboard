# 📊 Ecommerce Sales Analysis Dashboard

## 📌 Overview

This project analyzes ecommerce sales data to identify trends in revenue, customer behavior, and product performance. The goal is to derive insights that can help improve business decisions.

---

## 🛠 Tools Used

* Google Looker Studio
* MySQL Workbench
* SQL

---

## 🗄️ Dataset

The dataset includes:

* Order details (Order ID, Date)
* Product and category information
* Customer details
* Price and quantity

## ⚙️ SQL Analysis

Key queries performed:

* Total revenue calculation
* Monthly sales trend
* Top-selling products
* Category-wise performance

(SQL scripts are available in the `sql/` folder)

---

## 🧠 Key Insights

* Certain categories generate the highest revenue
* Sales show trends across different months
* A small set of products contributes most of the revenue
* Customer behavior varies across regions

---

## 🚀 Conclusion

This project helps in understanding ecommerce performance and supports better decision-making in pricing, marketing, and inventory management.
